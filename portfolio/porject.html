영동군청 기존 홈페이지의 데모사이트를 구축하였습니다. (랜딩페이지)

코드 간결화를 위해 효과를 템플릿화하여 적용했습니다.
CLASS를 이용하여 웹사이트 내 동일하게 사용되는 CSS, JS 효과를 템플릿으로 만들어 적용했습니다.
display:grid를 사용하여 레이아웃을 구현하였습니다.
하나의 grid-template을 사용하여 비슷한 형태의 하위 아이템들의 배치를 자유롭게 제어했습니다.
내용입력
내용입력
JS(JQuery)까지의 교과과정을 모두 끝낸 후 처음으로 작업한 프로젝트였습니다.
이 패이지를 만들면서 배운것은 구조를 고려하여 실용적으로 설계하는 방법입니다. 그동안 코드를 설계할 때 큰 구조를 생각하지 못하고 헤더, 본문, 푸터 등 위에서부터 순차적으로 설계를 하다보니, 코드를 재활용하지 못하고 같은 코드를 여러번 작성하고 있다는 것을 발견했습니다. 그래서 이미 완성은 했지만, 깨달은 점을 적용해보고 싶어, 섹션의 한 부분(여러 슬라이드로 구성된 '분야 별 정보')을 다시 설계해봤습니다.
각 슬라이드의 내용을 순차적으로 채워나가는 것이 아닌, 전체 슬라이드를 큰 구조부터 세부 구조 순으로 동시에 만들어 나갔습니다. 또 동일한 스타일과 애니메이션을 템플릿화하여 사용했습니다. 이렇게 하니, 코드가 조금은 더 읽기 쉬워지고 코드의 길이도 확연하게 줄일 수 있다는 것을 알게되었습니다. 이 경험은 코드를 설계할 때 단순히 화면을 구현하는 것에 만족하지 않고, 설계와 유지보수 모두에 더 실용적인 코드를 작성하는 방법을 계속 연구해보는 계기가 되었습니다.
마이데이터
마이데이터 기존 홈페이지의 데모사이트를 구축하였습니다. (랜딩페이지+서브페이지2)
HTML CSS JQuery
Frontend / 100%
Web
연속 선택 메뉴가 실제로 작동하도록 기능을 재현하였습니다.
이전, 다음 버튼으로 상위 또는 하위 메뉴로의 이동이 자유롭고, 메뉴탭 이동시 타이틀에 상위 메뉴에서 선택한 항목이 자동으로 나열됩니다.
그 밖에 항목 선택을 가이드하는 경고창 기능, 선택항목 종류에 따른 애니메이션 등 모든 기능을 최대한 재현하였습니다.
유지보수가 용이하도록 입력값을 끌어와 자동화하였습니다.
서브 페이지의 네비게이션 내용을 GNB에서 가져오고, 제목을 네비게이션에서 끌어오는 등 유지보수시에 손이 덜 갈 수 있도록 코드를 작성하였습니다.
더 부드럽고 자연스러운 인터렉션을 위해 addClass를 사용하였습니다.
제이쿼리로 애니메이션을 주게 되면 천천히 동작하고, 복잡한 동작을 주게 되면 버벅일 확률이 높기 때문에 자바스크립트로 애니메이션을 주는 일을 지양하고 CSS transiton을 주로 사용하였습니다.
이 프로젝트에서는 배너와 연속선택메뉴를 만들면서 배운 것이 많았습니다.
첫번째로 배너를 만들면서 Open API를 적용하는 법을 연습할 수 있었습니다. 한 페이지에 같은 API를 여러개 넣는 방법, 기본 스타일을 원하는 스타일로 수정하는 방법 등을 익힐 수 있었습니다. 또한 복잡한 애니메이션이 적용된 배너를 만들면서 작동 순서를 고려하여 애니메이션을 적용하는 법을 익힐 수 있었습니다.
두번째로 연속선택매뉴를 실제로 작동하게 구현해보면서 페이지간의 연결을 고려한 코드 디자인을 해보게 되었습니다. 이 경험을 통해 프론트엔드 지식만으로도 간단한 게임이나, 테스트 등 다양한 프로그램을 만들어볼 수 있겠다는 생각에 흥미가 더해졌습니다.
마이포트폴리오
저의 작업을 담은 포트폴리오 페이지를 구축하였습니다.
HTML CSS JQuery AdobeXD
Desgin / 100% Frontend / 100%
Responsive
미디어쿼리를 이용해 반응형 사이트를 구축하였습니다.
디바이스의 너비를 기준으로 콘텐트 구조를 달리하고, fine cusor와 hover기능 유무에 따라 애니메이션을 달리 적용하였습니다.
root를 사용하여 유지관리가 용이하도록 했습니다.
동일하게 적용되어야 할 컬러, 폰트 등의 스타일을 root로 지정하여 사용하여 유지관리에 용이할 수 있도록 했습니다.
동일한 엘리먼트의 입력을 자동화하고 내용입력 구간을 따로 만들어 유지관리에 용이하도록 했습니다.
동일한 구조를 가진 엘리먼트의 반복 입력으로 길어진 코드를 줄이고 싶었습니다. 자바스크립트로 반복되는 엘리먼트를 append하여 자동으로 입력될 수 있게 만들었습니다. 그리고 가변적으로 입력해야하는 내용을 관리할 수 있는 엘리먼트를 따로 만들어 입력한 내용을 자동완성 함수에 연결하였습니다.
처음으로 만든 반응형 사이트입니다. 반응형 사이트 제작에 대해서는 스스로 학습해야 했기 때문에 제작하는데 시간이 꽤 오래 걸렸습니다. 그러나 스스로 찾아 공부한만큼 진정으로 이해하면서 진도를 나갈 수 있었고, 공부한 내용을 블로그에 정리하기 시작한 계기도 되었습니다.
직접 디자인한 사이트라는 장점을 이용하여 부담없이 디자인을 변경해가며 공부하면서 추가로 알게 된 속성(root나 flex-grow 등)을 적용해보기도 했습니다. 또 내용 입력을 관리하는 엘리먼트를 따로 추가하는 등의 실험적인 방법도 시도해볼 수 있어 흥미로운 작업이었습니다.
